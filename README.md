# Combat Achievement Ironlock

A single-page OSRS-inspired HUD for an Ironman variant where Combat Achievements are the only currency to unlock skills, quests, and shops.

## Features
- HUD-style layout that keeps tasks, unlocks, and the rune journal visible without scrolling.
- Tier filter and completed toggle to quickly sort Combat Achievement cards and cash in points.
- Unlock console with segments for skills, quests, and shops; skill costs start at 1 point and grow by +1 per purchase.
- Rune Journal that logs completions, unlocks, manual point adjustments, and resets.
- Quick CA point modal and toast feedback to mirror an in-game companion feel.

## Running locally
Open `index.html` in your browser. No build tools are required.
